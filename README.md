Репозиторий с тестами к курсу Stepik'а "Python и Selenium". Ссылка на курс: https://stepik.org/course/575/syllabus .
В папке код теста, проверяющего наличие кнопки добавления в корзину на сайте маркетплейса, с проверкой наличия надписи на испанском языке.
Из РФ сайт недоступен, поэтому реализована возможность посещения через proxy с авторизацией. 
Данные для proxy и авторизации надо добавить в файл .env, БЕЗ КАВЫЧЕК! Запуск теста осуществляется командой pytest --language=es test_items.py
